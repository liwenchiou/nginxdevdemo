踩坑：開源版 Nginx 的致命傷

天下沒有白吃的午餐，免費版的 Nginx 藏著一個巨大的地雷。

⚠️ DNS 永久快取

Nginx 啟動時會把容器的 IP 死死記在記憶體裡。

當 Docker 容器當機重啟、獲得全新 IP 時，Nginx 還是會傻傻地把流量往舊 IP 丟，導致系統癱瘓！

動態服務發現：自動化解法

引入 docker-gen 打造企業級零感知熱更新。

👀 1. 即時監聽

24 小時緊盯 Docker Socket 的容器生命週期事件。

✍️ 2. 動態重寫

抓出最新存活的容器 IP，毫秒級重寫 upstreams.conf。

🔄 3. 無縫重載

發送 SIGHUP 訊號，Nginx 不斷線 Reload 生效！
